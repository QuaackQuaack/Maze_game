# Maze game
A maze game using turtle.

## Requirements
First you need to install tkinter
+ For Ubunutu
``` sudo apt-get install python3-tk```
+ For Debian 
```sudo dnf install python3-tkinter```

## Installation 
``` 
git clone https://github.com/QuaackQuaack/Maze_game.git 
```
